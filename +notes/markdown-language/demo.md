## Списки

### Маркированный список

* Первый элемент
* Второй элемент

  * Вложенный
  * Ещё один
* Третий элемент

### Нумерованный список

1. Первый
2. Второй
3. Третий

## Код

### Встроенный

`printf("Hello, world!");`

### Блок кода (C)

```c
#include <stdio.h>

int main()
{
    printf("Hello, Markdown!\n");
    return 0;
}
```

### Блок кода (Python)

```python
for i in range(3):
    print("Markdown test", i)
```
